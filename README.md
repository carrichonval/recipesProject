Projet de recettes

```
- Authentifications avec JWT
- Dashboard (Nombre de recettes sur le site, nombre de recette réalisés sur le site, nombre de recette personnel, nombre de recette personnels réalisés.. )
- Page de création de recette ( liste ingrédients et es étapes )
- Page de recherche de recette avec filtres ( type: entrée/plat/dessert, les mieux notés, les plus réalisés, les plus récentes, ... )
- Page de recette en direct ( les étapes s'affichent les une après les autres )
- List des utilisateurs ( avecle nombre de reecettes et leur moyenne de note qu'ils ont, et la possibilité de voir leurs recettes)
- Page de profi utilisateur ( bio, préférence de recette, note moyenne, nombre de recette, etc ..)
- Page de mofidication des recettes
- Feed à la instagram contenant des images ou vidéos ( avec comeentaires et likes) des résultats des recettes posté par les utilisateurs
- Possibilité de donné une note à la recette d'un autre persone
- Responsive mobile / pc / tablet
```
